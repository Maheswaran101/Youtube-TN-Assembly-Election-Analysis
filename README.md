# YouTube Data Analysis — Tamil Nadu Assembly Election 2026

## Overview
Analysis of the top 100 YouTube videos related to "Tamil Nadu Assembly" using YouTube Data API v3. Extracts video metadata and public comments to understand political discourse ahead of the 2026 Tamil Nadu Legislative Assembly Elections.

## What's Inside
- `Youtube_Data_Analysis.ipynb` — Full analysis notebook
- `TN_Assembly_Analysis_Report.pdf` — Summary presentation

## Key Findings
- DMK leads with 484 mentions (35.6%) and highest positive sentiment (33.7%)
- Vijay (TVK) is the most mentioned leader with 131 comments despite being a new entrant
- BJP faces the most negative sentiment in Tamil Nadu context
- 62.2% of comments are neutral — largely due to Tamil/Tanglish text

## Tools Used
- YouTube Data API v3
- Python (Pandas, NumPy, TextBlob, NLTK, Matplotlib, Seaborn)
- Jupyter Notebook

## How to Run
1. Clone the repo
2. Install dependencies: `pip install pandas textblob nltk matplotlib seaborn requests`
3. Replace `API_KEY` in the notebook with your YouTube Data API key
4. Run all cells in `Youtube_Data_Analysis.ipynb`
